# Rubiks-Cube

A java application which visualizes a virtual Rubik's cube. 

Faces are visualized as though the cube was an unfoleded box:


       |     |
       |  6  |
       |     |
       -------
       |     |
       |  4  |
       |     |
|     ||     ||     |
|  2  ||  1  ||  3  |
|     ||     ||     |
       |     |
       |  5  |
       |     |
